# Face-Recognition-using-Yolov

🎯 Face Detection Using Custom YOLOv8 (Video → Dataset → Training → Detection)

This project builds a custom face detection system using YOLOv8.
A video is converted into frames, annotated in YOLO format, used to train a model, and then applied back to the video to detect faces.

📌 Overview

Pipeline:

Video → Frames → Annotations → Dataset → Training → Evaluation → Detection Output

✨ Features
Extract frames from video
Prepare YOLO-format dataset
Train custom YOLOv8 model
Validate trained model
Detect faces in video
Save annotated output video
🧠 Tech Stack
Python
OpenCV
Ultralytics YOLOv8
File handling & dataset processing
